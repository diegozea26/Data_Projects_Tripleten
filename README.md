# Data_Projects_Tripleten
Projects developed during the **Business Intelligence Analyst Bootcamp at Tripleten**.  


| Project_number | Title | Description | 
| :--------------: | ---------- | -------------- |
| 1 | Manhattan Vacation Rental Market | The project task was to analyze Airbnb data to generate insights and support investment decisions related to property types. |
| 2 | Exploring Startups Trends with SQL | The project task was to analyze startup data using SQL to explore trends, patterns, and key metrics across the startup ecosystem. |
| 3 | E-commerce Company transactions | The project task was to analyze Performed product and user behavior for an e-commerce company to support data-driven decision-making. |
| 4 | Saving SuperStore | The project task was to analyze SuperStore performance data to identify profit and loss drivers, advertising impact, and return-related issues. |
| 5 | Returned Orders at the SuperStore | Built an interactive dashboard to analyze return patterns at SuperStore and identify root causes behind high return rates across different periods of the year |
